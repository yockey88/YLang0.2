\doxysubsection{Detailed Description}
This is the raw token, which contains all the information about a token before it is converted to a \mbox{\hyperlink{structylang_1_1Token}{Token}} object. \begin{DoxyNote}{Note}
The \mbox{\hyperlink{structylang_1_1RawToken}{Raw\+Token}} represents what is usually thought of as a token, but in the ylang compiler more detailed information is needed, so the \mbox{\hyperlink{structylang_1_1Token}{Token}} struct is used instead. 
\end{DoxyNote}
